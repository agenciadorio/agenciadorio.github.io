== Frequently Asked Questions ==

= After Updating, the Plugin Dashboard Looks Ugly =
 
If you just Updated to Version 2.5.0 then this is a Browser Cache issue. Please clean your Browser's Cache. The update includes many CSS changes.

= Is Defer Booster Code Compatible with my Theme? =
 
Yes. We tested the plugin on several Themes. Is 99.9% Compatible with any Theme.

= What is HeartBeat? =

As the name indicates the heartbeat API sends continuous pulse to perform periodic tasks. All hosting companies follow a general policy of sending a warning email when the CPU utilization of the hosting server exceeds certain limit. WP Speed Grades Lite Changes HeartBeat Pulse to from 15 to 60 Seconds. This will help to reduce the server load and save hosting cost.
