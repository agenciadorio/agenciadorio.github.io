A new plugin to customize title of a page/post is easier than ever!

== Description ==

The plugin will help you to create custom title for the posts and pages. Currently we need to add codded style to the page and posts title to setup a custom title. 
By installing custom-title plugin the admin can easily add the style to the heading. The admin can manage the plugin

A non-programmer can easily customize the title of their page/post in wordpress

After activating the plugin, a visual editor will be displayed in place of page / post title 


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `custom-title.php` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. We will get a custom editor for title in the pages/posts.

= Does the plugin support every version of wordpress =
