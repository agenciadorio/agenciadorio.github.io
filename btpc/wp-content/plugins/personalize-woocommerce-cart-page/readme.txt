== Description ==
It's a woocommerce extension which allow store owner to add text in every area of cart page. Following is list of area where these text/message can be displayed on cart page:

[vimeo https://vimeo.com/104847741]

<h3>Features</h3>
<ol>
	<li>Before cart table</li>
	<li>Before cart contents</li>
	<li>Cart contents</li>
	<li>Sfter cart contents</li>
	<li>After cart table</li>
	<li>After cart</li>
	<li>Proceed to checkout</li>
	<li>Cart coupon</li>
	<li>After cart totals</li>
	<li>Before cart totals</li>
	<li>Cart is empty</li>
	<li>Before mini cart (cart in widget)</li>
	<li>Widget shopping cart before buttons</li>
	<li>After mini cart</li>
	<li>Cart totals before shipping</li>
	<li>Cart totals after shipping</li>
	<li>Cart totals before order total</li>
	<li>Cart totals after order total</li>
	<li>Before shipping calculator</li>
	<li>After shipping calculator</li>
</ol>

<h3>Following button title/labels can be change</h3>
<ol>
	<li>Add to cart (Single product)</li>
	<li>Add to cart (Shop/Loop)</li>
	<li>Out of stock (Single product)</li>
	<li>View product (Grouped product - Shop/Loop)</li>
	<li>Select options (Variable products - Shop/Loop)</li>
	<li>Buy product (External product - Shop/Loop)</li>
	<li>Place order (Checkout page)</li>	
</ol>

<h3>Best of this plugin</h3>
All of these features of this plugin is using Woocommerce registered hooks. No need to change any of the file in woocommerce. Just install and play.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `NM WOOStore` menu

== Frequently Asked Questions ==

= DO I need to make changes in file? =

No
= How to access settings =
In Dashboard, you will menu title 'Woo Customizer'

= What version of Woocommerce should I have? =
You should have latest version.
