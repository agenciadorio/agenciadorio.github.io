A flexible and responsive menu management plugin, not just normal sticky menu. Compatible with any WordPress theme

== Description ==

Use this plugin to display a sticky menu to your WordPress site. It's not just normal a fixed menu on the header, instead when you scroll down, it will fade out, but when you scroll up, it will appear on the header. 

Plugin homepage: [http://ptheme.com/item/wp-sticky-menu/](http://ptheme.com/item/wp-sticky-menu/ "WP Sticky Menu")

= A few awesome features: =

*   Fully Responsive
*   Custom Logo Image
*   Facebook/Twitter social profiles
*   Live customizer to control the appearance
*   Easy to use

This plugin should work good with most of themes. It has been tested in Twenty Twelve, Twenty Thirteen, Twenty Fourteen, Twenty Fifteen, Twenty Sixteen and also other most popular WordPress themes. 

= Usage = 

After installing and activating this plugin, just assign a menu to WP Sticky Menu via `Appearance > Menus`. 
Configure the appearance of the WP Sticky Menu from `Appearance > Customize > WP Sticky Menu`

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `wp-sticky-menu` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. ‘WP Sticky Menu’ Option will be now available in Customizer
4. Assign a menu to WP Sticky Menu via `Appearance > Menus`.
